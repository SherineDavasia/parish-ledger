# Parish Ledger - Church Certificate Management System

A modern web application for managing Baptism, Marriage, and Death certificates for your church members.

## Features

- **Dashboard**: Overview of all certificates and statistics
- **Baptism Records**: Manage baptism certificates
- **Marriage Records**: Manage marriage certificates
- **Death Records**: Manage death certificates
- **Settings**: Configure application settings

## Tech Stack

- **Vite** - Fast build tool
- **React** - UI library
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first styling
- **shadcn/ui** - UI components
- **React Router** - Client-side routing
- **React Query** - Data fetching

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
